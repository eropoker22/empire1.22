# EMPIRE STREETS — KNIHOVNA DRBŮ

## Styl a pravidla

Tato knihovna je psaná jako temný cyberpunkový neo-noir: úsečná, černohumorná, akční, místy poetická a paranoidní. Drby mají hráče navést, ne mu dát hotovou odpověď.

### Co drby nikdy nesmí prozradit
- přesný počet obyvatel, jednotek nebo obránců,
- přesnou sílu útoku či obrany,
- přesné množství peněz, materiálů nebo zbraní,
- přesný čas plánované akce,
- přesný cíl připravovaného útoku,
- konkrétní složení výzbroje,
- existenci nebo typ pasti,
- interní systémová procenta a šance na úspěch.

### Úrovně spolehlivosti
- `confirmed` — veřejně ověřitelná událost nebo následek, který už proběhl.
- `credible` — silný signál; jádro informace bývá pravdivé, detaily mohou být zkreslené.
- `suspicion` — podezřelý signál s několika možnými vysvětleními.
- `rumor` — nepotvrzená fáma, často jen ozvěna skutečné události.
- `false_possible` — může být pravdivá, přehnaná nebo úplně chybná.

# 16. HLAVNÍ IMPLEMENTAČNÍ PRAVIDLO

> Drb má říct hráči, kam se podívat. Špehování mu má říct, co tam skutečně je.

Každý drb musí vzniknout z bezpečně zredukované projekce skutečné herní události. Textová vrstva nikdy nesmí dostat přístup k přesným tajným hodnotám, které nesmí být zveřejněny. Bezpečné je předávat jen předem odvozené kategorie, například:

```ts
{
  districtId,
  zone,
  actorVisibility: "hidden" | "player" | "alliance",
  intensityBand: "low" | "medium" | "high",
  direction: "increasing" | "decreasing" | "stable",
  ageBand: "fresh" | "recent" | "stale",
  confidence,
}
```

Zakázané je předávat šablonám:
- raw troop count,
- exact defense power,
- exact weapon inventory,
- exact attack target,
- exact command timing,
- trap state,
- private spy result,
- hidden economy values.
